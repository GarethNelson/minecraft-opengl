List of Implemented Features

I believe that I have implemented the following 11 Additional features:
  • Required Functionality:
    • 3D Perspective Viewing and Objects
    • Lighting and Smooth Shading
    • User Input
    • Computer Control over some elements of the scene.
  • Advanced Features:
    • Texture Mapping
    • On-screen Control Panel
    • Procedural and Physically Based Modeling
    • Collision Detection
    • Advanced Image-Based Techniques (Billboarding)
    • Sound
    • Game-Level Editor
    • Level of Detail Control
    • Multipass Rendering Effects
    • Occlusion Culling
    • View Frustum Culling
